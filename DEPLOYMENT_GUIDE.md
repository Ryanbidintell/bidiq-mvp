# BidIntell Deployment Guide

## Quick Start (5 minutes)

### 1. Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Login to Vercel
vercel login

# Deploy (from project root)
vercel --prod
```

### 2. Set Environment Variables

In Vercel Dashboard, add these secrets.

### 3. Update app.html

Replace hardcoded API calls with apiClient.js

### 4. Test and launch!
